Searching in Everyday Life and Computer Science

Everyday Life

  • Searching is a common occurrence in everyday life.
  • Examples:
    • Searching for a book in the library
    • Searching for a subscriber's telephone number in the telephone directory
    • Searching for one's name in the electoral rolls

Computer Science

  • In computer science, the problem of search has assumed enormous significance.
  • It spans a variety of applications and disciplines, including:
    • Searching for a key in a list of data elements
    • Searching for a solution to a problem in its space
    • Searching for patterns (images, voice, text, hyper text, photographs, etc.) in a repository of data or patterns

Types of Sort Algorithms

  • Internal Sort

    Sorting algorithms that sort data within the main memory of a computer.

  • External Sort

    Sorting algorithms that sort data that is too large to fit in main memory, using external storage devices such as hard drives.

  • Comparison Sort

    Sorting algorithms that compare elements to determine their order.

  • Non-Comparison Sort

    Sorting algorithms that do not compare elements, instead using other methods such as counting or hashing.

Common Sort Algorithms

  • Insertion Sort

    A simple, efficient sorting algorithm that inserts each element into its proper position in the sorted portion of the data.

  • Merge Sort

    A efficient, divide-and-conquer sorting algorithm that divides the data into smaller chunks, sorts each chunk, and then merges the sorted chunks back together.
